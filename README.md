## Installation
To install the required dependencies, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/smaruf/go-lang-study.git
    cd go-lang-study
    ```

2. Install Go (if not already installed):
    ```sh
    https://golang.org/doc/install
    ```
